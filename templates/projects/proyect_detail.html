{% if language == "es" %} Acerca del proyecto {% else %} About the Project {% endif %}

{% if language == "es" %} FitoMenu es una aplicación web desarrollada para el restaurante local Pollo Fito, con el objetivo de modernizar y digitalizar su proceso de atención al cliente y gestión interna. El sistema permite a los clientes acceder al menú mediante un código QR desde su dispositivo móvil, visualizar productos disponibles y realizar pedidos en tiempo real. Internamente, el personal de cocina recibe estos pedidos a través de una interfaz web para gestionarlos de manera más ágil y ordenada. {% else %} FitoMenu is a web-based solution designed to optimize the management experience for small restaurants. It provides essential features like digital QR menus, real-time order handling, a visual kitchen dashboard, and an intuitive admin panel. The system is lightweight, responsive, and easy to adapt to any restaurant setting. {% endif %}

{% if language == "es" %} Este proyecto surgió de la necesidad de optimizar los tiempos de atención, reducir errores manuales y mejorar la experiencia del cliente. La solución se diseñó considerando las particularidades de un micronegocio, permitiendo una implementación gradual, intuitiva y sostenible en el tiempo. {% else %} FitoMenu is a web-based solution designed to optimize the management experience for small restaurants. It provides essential features like digital QR menus, real-time order handling, a visual kitchen dashboard, and an intuitive admin panel. The system is lightweight, responsive, and easy to adapt to any restaurant setting. {% endif %}

{% if language == "es" %} Desafíos y soluciones {% else %} Challenges and Solutions {% endif %}

{% if language == "es" %} Sincronización de Pedidos en Tiempo Real {% else %} Real-Time Order Sync {% endif %}

{% if language == "es" %} Desafío: Lograr una comunicación fluida entre el menú del cliente y el sistema de cocina sin retrasos.
Solución: Se diseñó un flujo asincrónico de pedidos con lógica clara de estado y actualizaciones inmediatas, usando vistas bien optimizadas para garantizar una experiencia sin interrupciones. {% else %} Challenge: Achieve seamless communication between the customer menu and kitchen system without delays.
Solution: An asynchronous order flow was designed with clear state logic and immediate updates, using well-optimized views to ensure a smooth experience. {% endif %}

{% if language == "es" %} Presentación de Información Estratégica {% else %} Strategic Information Display {% endif %}

{% if language == "es" %} Desafío: Incorporar métricas útiles para la toma de decisiones en una interfaz amigable para usuarios no técnicos.
Solución: Se desarrolló un dashboard administrativo que presenta productos más vendidos por períodos, utilizando agregaciones en consultas SQL y diseño visual claro con Bootstrap. {% else %} Challenge: Incorporate useful metrics for decision-making in a user-friendly interface for non-technical users.
Solution: An admin dashboard was developed showing top-selling products per period, using SQL aggregations and a clean visual design with Bootstrap. {% endif %}

{% if language == "es" %} Modularización del Proyecto {% else %} Project Modularization {% endif %}

{% if language == "es" %} Desafío: Mantener el código escalable y fácil de mantener ante futuras ampliaciones.
Solución: Se estructuró el proyecto respetando la separación de responsabilidades: modelos, formularios, vistas, plantillas y lógica de negocio, siguiendo las mejores prácticas recomendadas por la documentación oficial de Django. {% else %} Challenge: Keep code scalable and easy to maintain for future expansions.
Solution: The project was structured with clear separation of concerns (models, forms, views, templates, business logic) following Django’s official best practices. {% endif %}

{% if language == "es" %} Capacitación y Usabilidad para el Personal del Restaurante {% else %} Training and Usability for Restaurant Staff {% endif %}

{% if language == "es" %} Desafío: Asegurar la adopción tecnológica por parte del personal sin conocimientos técnicos.
Solución: Se implementó una interfaz simple, basada en Bootstrap, acompañada de un manual de usuario y una sesión de capacitación presencial para asegurar el correcto uso del sistema. {% else %} Challenge: Ensure technology adoption by staff with no technical background.
Solution: A simple Bootstrap-based interface was implemented, supported by a user manual and a live training session to ensure proper usage. {% endif %}

{% if language == "es" %} Adaptabilidad y Rendimiento Móvil {% else %} Mobile Adaptability and Performance {% endif %}

{% if language == "es" %} Desafío: Garantizar una experiencia rápida y funcional en dispositivos móviles con conexión limitada.
Solución: Se priorizó la carga ligera del frontend, se aplicaron principios de diseño móvil-first, y se optimizó cada vista para minimizar consultas y carga innecesaria. {% else %} Challenge: Ensure a fast and functional experience on mobile devices with limited connectivity.
Solution: Lightweight frontend loading was prioritized, mobile-first principles were applied, and each view was optimized to reduce queries and unnecessary load. {% endif %}
